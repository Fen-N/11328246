### **Part 1 遊戲簡介**
本專案由pygame製作，詳細系統資訊可至 [requirements.txt](requirements.txt) 查看。  
本專案是想設計一個簡易的射擊小遊戲，只要點擊滑鼠就能射出子彈攻擊不斷移動的ufo，同時角色本身也可以WASD移動來躲避敵人射來的砲彈，一旦角色的3顆心都用光則遊戲結束，反之成功打敗兩個ufo則獲勝。
